static PointPlaneDistance(pt, plane_origin, plane_normal)

Compute the shortest distance from a point to a plane.

The distance is calculated as the perpendicular (orthogonal) distance from a point in 3D space to a plane defined by a point on the plane and its normal vector.

Parameters:

  • pt (array-like of float) – Coordinates of the point (x, y, z).

  • plane_origin (array-like of float) – A point lying on the plane (x0, y0, z0).

  • plane_normal (array-like of float) – Normal vector of the plane (nx, ny, nz).

Returns:

Perpendicular distance from the point to the plane.

Return type:

float

Notes

The distance is computed as:

d = |(P - P0) cdot n| / ||n||

where: - P is the point - P0 is a point on the plane - n is the plane normal vector

static PlaneAsEquation(plane_origin, plane_normal)

Convert a plane defined by a point and a normal vector into the standard plane equation form: ax + by + cz + d = 0.

The plane is defined using a point on the plane A(x1, y1, z1) and a normal vector N(a, b, c):

a(x - x1) + b(y - y1) + c(z - z1) = 0

Expanding gives:

ax + by + cz + d = 0 where d = -a*x1 - b*y1 - c*z1

Parameters:

  • plane_origin (array-like of float) – A point on the plane (x1, y1, z1).

  • plane_normal (array-like of float) – Normal vector of the plane (a, b, c).

Returns:

Coefficients [a, b, c, d] of the plane equation ax + by + cz + d = 0.

Return type:

list of float

Examples

>>> plane_origin = (5, 6, 2)
>>> plane_normal = (-1, 3, -7)

The plane equation is:

-1(x - 5) + 3(y - 6) - 7(z - 2) = 0

Expanding:

-x + 5 + 3y - 18 - 7z + 14 = 0 -x + 3y - 7z + 1 = 0

So the returned coefficients are:

[-1, 3, -7, 1]

IsPointOnPlane(plane_origin, plane_normal)

Check whether a point lies on a given plane within a numerical tolerance.

The function evaluates the perpendicular distance from the point to the plane and compares it against a predefined tolerance.

Parameters:

  • point (array-like of float) – Coordinates of the point to test (x, y, z).

  • plane_origin (array-like of float) – A point on the plane (x0, y0, z0).

  • plane_normal (array-like of float) – Normal vector of the plane (a, b, c).

Returns:

True if the point lies on the plane within tolerance, False otherwise.

Return type:

bool

Notes

A point is considered to lie on the plane if:

abs(distance) ≤ tolerance

where the distance is computed using the point-to-plane distance formula.

static SegSegIntersection(p1, p2, q1, q2, strict=True)

Check intersection between two 3D line segments (or infinite lines).

This function determines whether two segments (or lines) in 3D space intersect, and optionally restricts the intersection point to lie within the finite segment bounds.

Parameters:

  • p1 (array-like of float) – Endpoints of the first segment.

  • p2 (array-like of float) – Endpoints of the first segment.

  • q1 (array-like of float) – Endpoints of the second segment.

  • q2 (array-like of float) – Endpoints of the second segment.

  • strict (bool, optional) – If True, only intersections within both segments are considered (segment-segment intersection). If False, intersections on the infinite lines are considered (line-line intersection). Default is True.

Returns:

statusbool

True if a valid intersection exists, False otherwise.

pointarray-like of float or None

Coordinates of the intersection point if it exists, otherwise None.

Return type:

tuple (status, point)

Notes

The method uses a cross-product-based formulation to avoid solving linear systems explicitly. Numerical tolerance is controlled via Utilities.Tolerance.

Cases handled: - Parallel (no intersection) - Coincident (overlapping lines) - Skew (non-intersecting in 3D) - Proper intersection (unique point)

Examples

Two intersecting segments:

p1 = (0,0,0), p2 = (1,1,1) q1 = (0,1,0), q2 = (1,0,1)

Returns:

(True, [intersection_point])

Non-intersecting segments:

returns (False, None)

static VecAngle2D(u, v, radians=False)

Compute the signed angle between two 2D vectors.

The angle is measured from vector u to vector v using the atan2 determinant formulation, which preserves orientation (clockwise vs counterclockwise).

Parameters:

  • u (array-like of float) – First 2D vector (u1, u2).

  • v (array-like of float) – Second 2D vector (v1, v2).

  • radians (bool, optional) – If True, return the angle in radians. If False, return the angle in degrees. Default is False.

Returns:

Signed angle between the vectors: - in radians if radians=True - in degrees otherwise (range [0, 360))

Return type:

float

Notes

The angle is computed using:

angle = atan2(det(u, v), dot(u, v))

where:

dot = u · v det = u_x * v_y - u_y * v_x

This preserves directional information.

static VecAngleBetween(u, v)

Compute the unsigned angle between two vectors in 2D or 3D space.

The angle is always returned as the smallest positive angle between the two vectors, in degrees.

Parameters:

  • u (array-like of float) – First vector (2D or 3D).

  • v (array-like of float) – Second vector (2D or 3D).

Returns:

Angle between the vectors in degrees, in the range [0, 180].

Return type:

float

Raises:

ValueError – If either input vector has zero magnitude.

Notes

The angle is computed using the dot product formula:

cos(theta) = (u · v) / (||u|| ||v||)

followed by:

theta = arccos(clamped(cos(theta)))

where clamping ensures numerical stability in floating-point arithmetic.

static VecCrossProd(u, v)

Compute the cross product of two 3D vectors.

The cross product produces a vector that is orthogonal to both input vectors and follows the right-hand rule.

Parameters:

  • u (array-like of float) – First vector (u1, u2, u3).

  • v (array-like of float) – Second vector (v1, v2, v3).

Returns:

Cross product vector u × v.

Return type:

tuple of float

Notes

The cross product is defined as:

u × v = (u2*v3 - u3*v2, u3*v1 - u1*v3, u1*v2 - u2*v1)
