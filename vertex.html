Vertex

Core data structures for representing structural graphs, including nodes, edges, connectivity, and loads.

class Vertex.Vertex(X, Y, Z, id, label)

Bases: object

Initialize a vertex (node) in the structural graph.

Parameters:

  • X (float) – X-coordinate of the vertex.

  • Y (float) – Y-coordinate of the vertex.

  • Z (float) – Z-coordinate of the vertex.

  • id (int) – Unique identifier for the vertex.

  • label (int) – Label describing the role of the node. Typically used for boundary conditions or classification (e.g., support, free node).

Notes

A vertex represents a node in the structural graph. It stores both geometric information (coordinates) and topological relationships (adjacent nodes and incident edges), as well as state variables such as displacements.

AngleUniformity(nodes_2D)

Compute the Angle Uniformity Index (AUI) for a vertex.

The Angle Uniformity Index measures how evenly the edges connected to a node are distributed in 2D space. A value close to 1 indicates uniform angular spacing, while values closer to 0 indicate irregular angular distribution.

Parameters:

nodes_2D (dict or array-like) – 2D coordinates of all nodes in the graph. Must allow indexing as nodes_2D[node_id] -> (x, y).

Returns:

Angle Uniformity Index in the range [0, 1], where: - 1.0 indicates perfectly uniform angular distribution - 0.0 indicates highly non-uniform distribution

Return type:

float

Notes

  • Nodes with fewer than 2 connections return 1.0 by definition.

  • Angles are computed pairwise between adjacency vectors.

  • The method assumes 2D projections (x-y, x-z, or y-z plane).
