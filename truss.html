Truss

Core data structures for representing structural graphs, including nodes, edges, connectivity, and loads.

class Truss.Truss(nodal_coords, edges, weights, fixed_nodes=None, loads=None, load_labels=None)

property BasicMetrics

Basic structural metrics of the system.

Returns:

Dictionary of key structural metrics:

  • ”maxF”: maximum axial force

  • ”planarity”: planarity degree of the graph

  • ”maxL”: maximum element length

  • ”n_nodes”: number of nodes in the graph

  • ”minF”: minimum axial force

  • ”n_bars”: number of structural elements (edges)

  • ”minL”: minimum element length

  • ”static_action”: global static action of the system

Return type:

dict

EulerCriticalBucklingLimits(k, E, I)

Compute Euler critical buckling limits for all edges in the graph.

This method evaluates the critical buckling load for each edge using Euler’s buckling formula.

Parameters:

  • k (float) – Effective length factor (depends on boundary conditions).

  • E (float) – Young’s modulus of the material.

  • I (float) – Second moment of area (area moment of inertia).

Returns:

Critical buckling load values for each edge in the graph.

Return type:

list of float

Notes

  • Assumes each edge implements EulerCriticalBucklingLimit(k, E, I).

  • Results are returned in the same order as self.EdgeObjs.

ForceDirectionality(load_node_coords, support_node_coords)

Compute global force directionality of a truss structure.

This metric evaluates how well member axial forces align with the global load-transfer direction defined between load and support nodes.

Parameters:

  • load_node_coords (list of tuple (x, y, z)) – Coordinates of nodes where loads are applied.

  • support_node_coords (list of tuple (x, y, z)) – Coordinates of support nodes.

Returns:

  • dir_score (float) – Global directionality score in [0, 1], where higher values indicate better alignment between member forces and global load direction.

  • members_alignment (numpy.ndarray) – Array of cos(theta_i) values for each structural member.

  • global_dir_unit (numpy.ndarray) – Unit vector representing the global force transmission direction.

Raises:

ValueError – If the computed global direction vector has zero magnitude.

Notes

  • Uses axial force magnitudes as weighting factors.

  • Alignment is computed via cosine similarity between member direction and global load direction.

Compute the maximum absolute nodal displacement in the structure.

This method scans all vertex displacements in x, y, and z directions and returns the maximum absolute value.

Parameters:

cm (bool, optional) – If True, converts the result from meters to centimeters. Default is False.

Returns:

Maximum absolute displacement across all nodes and directions.

Return type:

float

Notes

  • Considers absolute values of Ux, Uy, Uz.

  • Useful for assessing global deformation magnitude.

MaxGlobalUtilization(A, yield_strength, percent=False)

Compute the maximum structural utilization across all members.

Utilization typically represents how close each member is to its allowable stress or capacity limit.

Parameters:

  • A (float) – Cross-sectional area of the members.

  • yield_strength (float) – Material yield strength.

  • percent (bool, optional) – If True, utilization is returned in percentage form. Default is False.

Returns:

Maximum utilization value among all members.

Return type:

float

Notes

  • Relies on self.Utilizations(A, yield_strength, percent).

  • Higher values indicate more critical structural demand.

Compute the total mass of the structure.

Parameters:

  • d (float) – Material density.

Returns:

Total mass of the structure.

Return type:

float

Notes

  • Assumes uniform density across all members.

  • Relies on self.Volume(A) for total structural volume.

Compute the mean structural utilization across all members.

Utilization represents how much of the allowable capacity each member is using. This method aggregates it into an average global indicator.

Parameters:

  • yield_strength (float) – Material yield strength.

  • percent (bool, optional) – If True, utilization is expressed in percentage form. Default is False.

Returns:

Mean utilization across all members.

Return type:

float

Notes

  • Based on self.Utilizations(A, yield_strength, percent).

  • Provides a global measure of structural demand distribution.

MinGlobalUtilization(A, yield_strength, percent=False)

Compute the minimum structural utilization across all members.

Utilization measures how much of the allowable capacity each member is using. This method returns the least-utilized member in the structure.

Parameters:

  • A (float) – Cross-sectional area of the members.

  • yield_strength (float) – Material yield strength.

  • percent (bool, optional) – If True, utilization is expressed in percentage form. Default is False.

Returns:

Minimum utilization value among all members.

Return type:

float

Notes

  • Based on self.Utilizations(A, yield_strength, percent).

  • Useful for identifying underutilized structural members.

UnconstrainedNodes(max_iter=100)

Propagate constrained (fixed) nodes and edges in a graph until equilibrium closure, and extract unconstrained substructures.

This method iteratively expands constraints starting from external nodes until no further propagation is possible.

Parameters:

max_iter (int, optional) – Maximum number of propagation iterations for safety. Default is 100.

Returns:

  • unconstrained_nodes (list of int) – Node IDs that are not constrained after propagation.

  • unconstrained_pairs (list of tuple of int) – Edge pairs connecting unconstrained nodes.

  • unconstrained_faces (list of list of int) – Faces composed entirely of unconstrained nodes.

Notes

  • Performs iterative constraint propagation over nodes and edges.

  • Stops when convergence is reached or max_iter is exceeded.

UpdateAnalyzedGraph(axial_forces, displacements, strain, stress)

Update graph elements with results from structural analysis.

This method assigns computed physical quantities (forces, displacements, strain, and stress) back to the corresponding edges and vertices of the graph.

Parameters:

  • axial_forces (list of float) – Axial force values for each edge.

  • displacements (list of (float, float, float)) – Nodal displacement vectors (Ux, Uy, Uz).

  • strain (list of float) – Strain values for each edge.

  • stress (list of float) – Stress values for each edge.

Notes

  • Assumes input arrays are ordered consistently with:

    • self.EdgeObjs

    • self.VertexObjs

  • Updates internal edge weights with axial forces.

Utilizations(A, yield_strength, percent=False)

Compute structural utilization for each member in the graph.

Utilization measures the ratio between axial force demand and material capacity.

Parameters:

  • A (float) – Cross-sectional area of members.

  • yield_strength (float) – Material yield strength.

  • percent (bool, optional) – If True, returns utilization as a percentage. Default is False.

Returns:

Utilization values for each edge in the graph.

Return type:

list of float

Notes

  • Uses axial forces stored in self.AxialForces.

  • Formula: U = |F| / (A * yield_strength)

Volume(A)

Compute the total volume of the structural system.

The volume is calculated as the sum of each member’s volume, assuming a constant cross-sectional area for all elements.

Parameters:

A (float) – Cross-sectional area of each member.

Returns:

Total structural volume.

Return type:

float

Notes

  • Assumes uniform cross-section across all members.

  • Uses self.Lengths, where each entry corresponds to an element length.
