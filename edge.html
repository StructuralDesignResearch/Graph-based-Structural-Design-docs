Edge

Core data structures for representing structural graphs, including nodes, edges, connectivity, and loads.

class Edge.Edge(start_node, end_node, id, weight, directed=False)

Bases: object

Initialize an edge in the structural graph.

Parameters:

  • start_node (int) – Index or identifier of the starting node of the edge.

  • end_node (int) – Index or identifier of the ending node of the edge.

  • id (int) – Unique identifier for the edge.

  • weight (float) – Weight associated with the edge. In structural contexts, this often represents axial force.

  • directed (bool, optional) – Indicates whether the edge is directed. If True, the edge has an orientation from start_node to end_node. Default is False.

Notes

This class represents a connection between two nodes in the graph. In truss systems, edges typically correspond to structural members (bars).

property StiffnessMatrix

Stiffness matrix of the edge element.

Returns:

6x6 stiffness matrix for a 3D truss element, computed based on the edge length and axial force. The matrix is defined in the local coordinate system of the edge.

Return type:

numpy.ndarray

Notes

The matrix must be multiplied with the stiffness coefficient (EA/L) to obtain the actual stiffness matrix for structural analysis. The current implementation assumes a unit stiffness coefficient for demonstration purposes.

EulerCriticalBucklingLimit(k, E, I)

Compute the Euler critical buckling load for the element.

The method evaluates the Euler buckling load for a structural member under compression. If the element is in tension (positive force), the buckling limit is set to zero.

Parameters:

  • k (float) – Effective length factor depending on boundary conditions.

  • E (float) – Young’s modulus of the material.

  • I (float) – Second moment of area (area moment of inertia) of the cross-section.

Returns:

Euler critical buckling load. Returns 0 if the element is in tension.

Return type:

float

Notes

The buckling load is computed as:

P_cr = π² * E * I / (k * L)²

where L is the edge length.
